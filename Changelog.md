# Changelog

本文件用于维护每个版本对外发布的更新说明。

编写约定：
- 每个版本使用 `## v版本号` 作为标题，例如 `## v0.3.6`
- 标题下方直接填写会展示在应用更新弹窗中的 Markdown 内容
- 使用 `---` 或下一个版本标题结束当前版本说明

## v0.7.3

### ✨ 新增功能
- 设置页新增 Windows WebView2 硬件加速开关，方便在不同显卡和驱动环境下自行选择更稳定的渲染方式

### 🐛 问题修复
- 修复逐条删除当前已加载历史记录后，分页偏移未同步更新，可能导致主面板长期停留在“正在加载历史...”的问题

---

## v0.7.2

### ✨ 新增功能
- 新增主面板搜索快捷键配置，可在设置页自定义或清空，默认保持 `Ctrl/Cmd + F` 聚焦搜索框
- 新增历史筛选切换快捷键配置，默认 `Ctrl + Tab` 在全部、置顶、文本、图片、图文筛选间循环切换，`Shift` 可反向切换

### 🚀 优化改进
- 开机自启动时统一保持后台常驻，只显示托盘图标，不再默认打开主面板
- 主面板隐藏或切换到其他应用后，会自动取消搜索框焦点，下次打开不再残留输入光标

---

## v0.7.1

### 🚀 优化改进
- 调整历史记录数字按键标识位置，现在显示在左上角粘贴来源图标右侧，列表信息更集中
- 刷新应用图标资源，优化不同平台和尺寸下的图标显示一致性

### 🐛 问题修复
- 修复全局快捷键或快速粘贴快捷键被其他程序占用时，应用可能直接启动失败的问题

---

## v0.7.0

### ✨ 新增功能
- 新增快速粘贴快捷键：按住快捷键可直接呼出主面板，重复按键切换候选项，松开后立即粘贴当前选中内容
- 主面板前 10 条历史记录新增数字按键标识，可通过 `Ctrl + 1` 到 `Ctrl + 0` 快速粘贴对应内容
- 历史记录现在支持拖拽使用，可将文本、链接、图片等内容从主面板拖到其他应用；图片会尽量以原生文件形式拖出，兼容更多目标应用
- 新增后台启动支持，配合开机启动时可静默常驻托盘，不再每次启动都打断当前工作

### 🚀 优化改进
- 优化主面板唤起、历史刷新和设置监听，减少重复查询与重复监听，历史较多时操作更轻快
- 优化设置页帮助提示，说明信息以悬浮提示展示，界面更清爽
- 更新应用图标资源，使不同平台和尺寸下的图标显示更一致
- 更新 README 与产品说明，补充快速粘贴、拖拽和后台启动等新能力

## v0.5.3

### ✨ 新增功能
- 历史记录标签支持更直接的行内操作
  - 鼠标悬停标签时可直接点击 `×` 移除标签
  - 当某个标签被全部移除后，会自动退出该标签筛选并恢复显示全部历史

### 🐛 问题修复
- 修复主面板在设置页返回后尺寸被意外放大的问题
- 修复主面板在单屏场景下粘贴完成后再次唤出时宽高轻微变大的问题
- 修复主面板在双屏不同缩放比例之间切换时，窗口尺寸被放大或缩小的问题
- 修复手机电脑互传会话超时后重新进入页面时无法自动创建新会话的问题
- 修复手机电脑互传页在会话超时后返回主面板时可能出现卡顿的问题
- 修复标签弹窗点击当前历史记录区域不会关闭的问题，现已改为点击非弹窗区域即关闭
- 修复复制音效在主面板隐藏、窗口失焦等场景下偶发不触发的问题

### 🚀 优化改进
- 去掉手机端互传文本消息下方多余的复制按钮，只保留消息内容展示
- 调整设置页中多个选项的帮助提示展示，减少界面噪音
- 将“标签名称”配置项从“历史”分组移动到“通用”分组
- 优化标签名称输入框字号与 placeholder 字号，使其与整体设置页更一致
- 更新 README 与 README.zh-CN，使文档描述与当前代码实现保持一致

## v0.5.0

### ✨ 新增功能
- 新增手机电脑互传，扫码后可以在电脑和手机之间收发文字、图片和文件
- 新增复制音效设置，复制时可以有声音反馈

- 剪贴板内容的捕获和复制反馈更稳定，使用时更少遇到内容缺失或反馈不明显的情况
- 设置项修改后会立即生效，不需要额外点保存

## v0.4.1

### 🐛 问题修复

### 🚀 优化改进
- 优化 Linux 直接粘贴能力检测，在桌面会话识别不明确时可根据已安装工具选择可用后端。
