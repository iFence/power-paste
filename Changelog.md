# Changelog

本文件用于维护每个版本对外发布的更新说明。

编写约定：
- 每个版本使用 `## v版本号` 作为标题，例如 `## v0.3.6`
- 标题下方直接填写会展示在应用更新弹窗中的 Markdown 内容
- 使用 `---` 或下一个版本标题结束当前版本说明

## v0.4.1

### 🐛 问题修复
- 修复 macOS 升级后已授权但直接粘贴仍提示缺少“辅助功能 / 自动化”权限的问题，降低自动化脚本失败对直接粘贴的影响。
- 修复 macOS 文本写入剪贴板时的兼容性问题，直接粘贴文本内容更稳定。
- 修复全局快捷键中 `Meta` 修饰键在 macOS / Linux 上显示和保存不一致的问题。
- 修复清空历史记录后仍可能重新捕获同一剪贴板内容的问题。
- 修复面板在显示、隐藏后最小尺寸状态可能丢失的问题。

### 🚀 优化改进
- 优化 Linux 直接粘贴能力检测，在桌面会话识别不明确时可根据已安装工具选择可用后端。

---

## v0.4.0

### ✨ 新增功能
- Linux 直接粘贴新增 Wayland 会话支持；安装 `wtype` 后即可在 Wayland 下直接发送粘贴快捷键，不再仅限于 `X11 + xdotool`。

### 🚀 优化改进
- 优化 Linux 下历史记录的来源应用识别逻辑。
